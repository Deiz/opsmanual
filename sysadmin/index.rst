How to get Sysadmin help
========================

-  Use one of our request webforms:

   -  **`Request a new
      service <https://docs.google.com/spreadsheet/viewform?formkey=dC1IZ2kwRDMtX2ZaM0ZELWVJQzBrZXc6MQ#gid=14>`__**
      (e.g. wordpress site or blog, mailing list, webapp, server)
   -  **`Report an issue with an existing
      service <https://docs.google.com/forms/d/1YLFdarGad0zoES2_doiIrgbe-kneHMQ3MPra9TiuUo8/viewform>`__**
      (general/unreachable/slowness etc issues)
   -  **`Request help with coding related
      issues <https://docs.google.com/forms/d/1E10sqF8GJgup8IpCR2CoNZrnyTffeZ-Almxq9hVe2UU/viewform>`__**
      (help with adding new plugin code, css modifications)

-  Ask your Unit's Sysadmin representative:

   -  Network: Neal Bastek
   -  LTP: Sander van der Waal
   -  Services: Joe Tsoi
   -  Knowledge: Tryggvi Bjorgvinsson

      -  A list of all projects, and there units, can be found on toggl:
         https://www.toggl.com/projects

-  Join our `IRC channel
   **#okfn** <http://webchat.freenode.net/#okfn>`__ on FreeNode.
-  Send an email to <**sysadmin** at **okfn** dot **org**>
